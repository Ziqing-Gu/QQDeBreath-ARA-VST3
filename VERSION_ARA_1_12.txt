QQDeBreath ARA/VST3 1.12
Date: 2026-07-11

- ARA Audio Sources preload into immutable memory on Load/project restore.
- Realtime playback uses memory instead of per-block Reaper ARA source reads.
- Multi-event Load caches every unique underlying source.
- Explicit Load reports preload failures instead of hiding them.
- ARA 1.11 defaults and ARA 1.10 realtime state-cache behavior retained.
- Native C++ analyzer and QQDeBreathTool 1.11 model unchanged.
