QQDeBreath 1.15 - macOS VST3 / AU installation and usage guide
================================================================

PACKAGE CHOICE
--------------

1. QQDeBreath-macOS-Apple-Silicon-1.15.zip
   VST3 for Apple Silicon Macs (M1/M2/M3/M4 and later).

2. QQDeBreath-macOS-Intel-1.15.zip
   VST3 for Intel Macs.

3. QQDeBreath-macOS-AU-Universal-1.15.zip
   Universal 2 Audio Unit for Logic Pro. It contains arm64 and x86_64.

SYSTEM REQUIREMENTS
-------------------

- macOS 11 or later.
- A compatible 64-bit DAW.
- These free builds are ad-hoc signed, not Developer ID signed or notarized.

INSTALL LOCATIONS
-----------------

Copy QQDeBreath.vst3 to:
  /Library/Audio/Plug-Ins/VST3/

Copy QQDeBreath.component to:
  /Library/Audio/Plug-Ins/Components/

Restart the DAW and rescan plug-ins after installation or replacement.

USAGE
-----

ARA in Cubase/Nuendo:
1. Add QQDeBreath as an Audio Extension to the selected audio event.
2. Open QQDeBreath and click Load.
3. Click Analyze, edit Breath/Noise regions, monitor the required stems, and export when ready.

Plain VST3 recording:
1. Insert QQDeBreath on the vocal track.
2. Click Record to arm it, then play the DAW timeline.
3. Stop DAW playback to stop recording.
4. Click Analyze and continue editing/monitoring.

Logic Pro / AU / ARA 2:
- Logic Pro does not load VST3. Install QQDeBreath.component.
- Insert QQDeBreath in the first Audio Effect slot to use ARA mode.
- On Apple Silicon, Apple currently requires Logic Pro to run with Rosetta for third-party ARA Audio Units. Quit Logic, open Applications, select Logic Pro, choose Get Info, enable Open using Rosetta, then reopen Logic.

FIRST-LAUNCH SECURITY NOTICE
----------------------------

If macOS blocks QQDeBreath during the first scan:
1. Trigger one plug-in scan in the DAW.
2. Open System Settings > Privacy & Security.
3. Find the QQDeBreath warning and click Open Anyway.
4. Enter the Mac login password and confirm.
5. Restart the DAW and rescan/reactivate QQDeBreath.

Trusted-source fallback commands:

  sudo xattr -dr com.apple.quarantine "/Library/Audio/Plug-Ins/VST3/QQDeBreath.vst3"
  sudo xattr -dr com.apple.quarantine "/Library/Audio/Plug-Ins/Components/QQDeBreath.component"

Only run these commands for archives downloaded from the official QQDeBreath GitHub repository.

VERSION 1.15
------------

1.15 fixes Breath Norm and Target resetting when the editor for the same plug-in instance is closed and reopened. Global defaults are now applied once per plug-in instance, while DAW project state, Global Gain, Global EQ, per-Breath Gain/EQ, and edited regions continue to restore normally. Analysis, embedded model data, monitoring DSP, EQ, Fade, Gain, and three-stem export audio are unchanged.
